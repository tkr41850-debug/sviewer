1. Create src/hooks/useKeyboardShortcuts.ts:

   The hook attaches a global `keydown` listener on `document` and dispatches actions based on the key pressed.

   **Input guard (per D-12):** Before processing any key, check if `document.activeElement` matches any of: `INPUT`, `TEXTAREA`, `SELECT` tagNames, or has attribute `contenteditable="true"`. If so, return early (no-op).

   **Shortcut handler map:**
   ```typescript
   const SHORTCUTS: Record<string, { action: () => void; toast: { key: string; action: string } | null }> = {
     'ArrowLeft':  { action: () => onPan('left'),  toast: { key: '\u2190', action: 'Pan left' } },
     'ArrowRight': { action: () => onPan('right'), toast: { key: '\u2192', action: 'Pan right' } },
     '+':          { action: () => onZoom('in'),   toast: { key: '+', action: 'Zoom in' } },
     '=':          { action: () => onZoom('in'),   toast: { key: '+', action: 'Zoom in' } },
     '-':          { action: () => onZoom('out'),  toast: { key: '-', action: 'Zoom out' } },
     'r':          { action: () => onReset(),      toast: { key: 'r', action: 'Reset zoom' } },
     't':          { action: () => cycleTheme(),   toast: { key: 't', action: 'Toggle theme' } },
     '?':          { action: () => toggleHelp(),   toast: null },
     'Escape':     { action: () => closeModals(),  toast: null },
   };
   ```

   **Hook signature:** `useKeyboardShortcuts(callbacks: { onPan: (dir: 'left'|'right') => void; onZoom: (dir: 'in'|'out') => void; onReset: () => void; }): void`

   - `onPan`, `onZoom`, `onReset` are callbacks provided by the caller (App.tsx) that manipulate chart view state. For Phase 5, these callbacks will update the URL state range. Arrows pan by 10% of current visible range (per D-13). +/- zoom 20% around center.
   - `cycleTheme`: Read current `viewState.themeOverride`, cycle `system -> light -> dark -> system`. Dispatch `SET_THEME_OVERRIDE` with next value.
   - `toggleHelp`: Dispatch `SET_HELP_MODAL` with `!viewState.helpModalOpen`.
   - `closeModals`: If `helpModalOpen` is true, dispatch `SET_HELP_MODAL` with `false`. (Escape also closes export dropdown, but that's handled internally by ExportDropdown.)
   - For each matched shortcut, if `toast` is not null, dispatch `SET_KEY_TOAST` with the toast payload.

   Use `useEffect` with `document.addEventListener('keydown', handler)` and cleanup.

   **Key toast auto-dismiss:** After dispatching SET_KEY_TOAST with a non-null value, set a 1000ms timeout to dispatch `SET_KEY_TOAST` with null. Clear previous timeout on new dispatch (per D-14: rapid successive shortcuts reset timer).
