Create `src/components/dashboard/SessionTimeline.tsx` (per D-07, D-08).

Props interface:
```typescript
import type { PostureRecord } from '../../data/types';

interface TimelineSegment {
  startTime: number;
  endTime: number;
  type: 'good' | 'slouch' | 'screenOff';
  durationMs: number;
}

interface SessionTimelineProps {
  records: PostureRecord[];
  thresholdPx: number;
  onSegmentClick?: (startTime: number, endTime: number) => void;
}
```

Implementation:

1. **Segment computation** — define a `computeSegments(records, thresholdPx)` function:
   - Walk through records sequentially
   - For each record, determine its state:
     - `isScreenOff === true` -> 'screenOff'
     - `deltaY !== null && Math.abs(deltaY) > thresholdPx` -> 'slouch'
     - else -> 'good'
   - Merge consecutive records with the same state into one segment
   - Each segment has `startTime` (first record's time), `endTime` (last record's time + one interval approximated as `endTime - startTime` of next record, or samplingInterval for last), `type`, `durationMs`
   - For endTime of the last record in a segment: use the next record's time if available, otherwise add the median interval between this segment's records (or 0 if single record)

2. **Rendering** — render a horizontal bar:
   ```tsx
   <div className="w-full" aria-label="Session timeline">
     <div className="flex items-center mb-2">
       <h3
         className="text-sm font-semibold uppercase tracking-wide"
         style={{ color: 'var(--color-text-secondary)' }}
       >
         Session Timeline
       </h3>
     </div>
     <div
       className="w-full h-8 rounded-md overflow-hidden flex"
       style={{ background: 'var(--color-surface)', border: '1px solid var(--color-border)' }}
     >
       {segments.map((seg, i) => (
         <button
           key={i}
           className="h-full transition-opacity hover:opacity-80"
           style={{
             width: `${(seg.durationMs / totalDuration) * 100}%`,
             backgroundColor: segmentColor(seg.type),
             cursor: onSegmentClick ? 'pointer' : 'default',
             minWidth: '1px',
           }}
           onClick={() => onSegmentClick?.(seg.startTime, seg.endTime)}
           title={`${seg.type === 'screenOff' ? 'Screen Off' : seg.type === 'slouch' ? 'Slouching' : 'Good Posture'} — ${formatSegmentDuration(seg.durationMs)}`}
           aria-label={`${seg.type} segment, ${formatSegmentDuration(seg.durationMs)}`}
         />
       ))}
     </div>
     {/* Legend */}
     <div className="flex gap-4 mt-2">
       <LegendItem color="oklch(65% 0.2 145)" label="Good" />
       <LegendItem color="var(--color-destructive)" label="Slouching" />
       <LegendItem color="oklch(60% 0 0)" label="Screen Off" />
     </div>
   </div>
   ```

3. **Color mapping**:
   - `good`: `oklch(65% 0.2 145)` (green)
   - `slouch`: Use CSS var `--color-destructive` — but since it's inline style, read it via the var: `var(--color-destructive)`. Actually for inline backgroundColor, use the oklch value directly since var() works in style prop: `'var(--color-destructive)'`
   - Wait — `style={{ backgroundColor: 'var(--color-destructive)' }}` works in React. Use that.
   - `screenOff`: `oklch(60% 0 0)` (gray — use oklch for consistent lightness in dark mode). Actually better: use `var(--color-border)` which is already theme-aware gray.

   Final color mapping function:
   ```typescript
   function segmentColor(type: 'good' | 'slouch' | 'screenOff'): string {
     switch (type) {
       case 'good': return 'oklch(65% 0.2 145)';
       case 'slouch': return 'var(--color-destructive)';
       case 'screenOff': return 'var(--color-border)';
     }
   }
   ```

4. **Duration formatter**:
   ```typescript
   function formatSegmentDuration(ms: number): string {
     if (ms < 60000) return `${Math.round(ms / 1000)}s`;
     if (ms < 3600000) return `${Math.round(ms / 60000)}m`;
     const h = Math.floor(ms / 3600000);
     const m = Math.round((ms % 3600000) / 60000);
     return `${h}h ${m}m`;
   }
   ```

5. **LegendItem** — simple inline component:
   ```tsx
   function LegendItem({ color, label }: { color: string; label: string }) {
     return (
       <div className="flex items-center gap-1.5">
         <div className="w-3 h-3 rounded-sm" style={{ backgroundColor: color }} />
         <span className="text-xs" style={{ color: 'var(--color-text-secondary)' }}>{label}</span>
       </div>
     );
   }
   ```

6. Compute `totalDuration` as `segments.reduce((sum, s) => sum + s.durationMs, 0)`. Guard against 0 (return empty div if no segments).

1. Create `src/components/dashboard/CalendarHeatmap.tsx` (per D-09, D-10):

Props interface:
```typescript
import type { PostureRecord } from '../../data/types';

interface HeatmapCell {
  hour: number;     // 0-23
  day: string;      // ISO date string "YYYY-MM-DD"
  slouchRate: number; // 0-100
  sampleCount: number;
}

interface CalendarHeatmapProps {
  records: PostureRecord[];
  thresholdPx: number;
}
```

Implementation:

a. **Data computation** — define `computeHeatmapData(records, thresholdPx)`:
   - Group active records (non-screen-off, non-null deltaY) by day and hour
   - Day: `new Date(record.time).toISOString().slice(0, 10)` (e.g., "2023-11-14")
   - Hour: `new Date(record.time).getHours()` (0-23)
   - For each (day, hour) cell: count total records and slouching records (abs(deltaY) > thresholdPx), compute slouchRate = (slouchCount / totalCount) * 100
   - Return: `{ cells: HeatmapCell[], days: string[], hours: number[] }`

b. **Color interpolation** — define `heatmapColor(slouchRate, sampleCount)`:
   - sampleCount === 0: return `'var(--color-surface)'` (empty cell)
   - slouchRate <= 20: `oklch(75% 0.2 145)` (green — great posture)
   - slouchRate <= 40: `oklch(80% 0.15 120)` (light green)
   - slouchRate <= 60: `oklch(80% 0.15 85)` (amber)
   - slouchRate <= 80: `oklch(70% 0.18 50)` (orange)
   - slouchRate > 80: `oklch(65% 0.22 25)` (red — heavy slouching)

c. **Rendering** — CSS Grid-based heatmap:
   ```tsx
   <div className="w-full" aria-label="Posture calendar heatmap">
     <h3
       className="text-sm font-semibold uppercase tracking-wide mb-2"
       style={{ color: 'var(--color-text-secondary)' }}
     >
       Posture by Hour
     </h3>
     <div className="overflow-x-auto">
       <div
         style={{
           display: 'grid',
           gridTemplateColumns: `80px repeat(24, minmax(28px, 1fr))`,
           gap: '2px',
         }}
       >
         {/* Header row — hour labels */}
         <div /> {/* empty corner cell */}
         {Array.from({ length: 24 }, (_, h) => (
           <div
             key={`h-${h}`}
             className="text-center text-xs"
             style={{ color: 'var(--color-text-secondary)' }}
           >
             {h === 0 ? '12a' : h < 12 ? `${h}a` : h === 12 ? '12p' : `${h - 12}p`}
           </div>
         ))}
         {/* Data rows — one per day */}
         {days.map(day => (
           <React.Fragment key={day}>
             <div
               className="text-xs truncate pr-2 flex items-center"
               style={{ color: 'var(--color-text-secondary)' }}
             >
               {formatDayLabel(day)}
             </div>
             {Array.from({ length: 24 }, (_, h) => {
               const cell = cellMap.get(`${day}-${h}`);
               const rate = cell?.slouchRate ?? 0;
               const count = cell?.sampleCount ?? 0;
               return (
                 <div
                   key={`${day}-${h}`}
                   className="rounded-sm aspect-square"
                   style={{
                     backgroundColor: heatmapColor(rate, count),
                     minHeight: '20px',
                   }}
                   title={count > 0 ? `${day} ${h}:00 — ${Math.round(rate)}% slouch (${count} samples)` : `${day} ${h}:00 — No data`}
                 />
               );
             })}
           </React.Fragment>
         ))}
       </div>
     </div>
     {/* Color legend */}
     <div className="flex items-center gap-2 mt-2">
       <span className="text-xs" style={{ color: 'var(--color-text-secondary)' }}>Good</span>
       {[0, 20, 40, 60, 80, 100].map((rate, i) => (
         <div
           key={i}
           className="w-4 h-4 rounded-sm"
           style={{ backgroundColor: heatmapColor(rate, 1) }}
         />
       ))}
       <span className="text-xs" style={{ color: 'var(--color-text-secondary)' }}>Poor</span>
     </div>
   </div>
   ```

d. For single-day data (per D-09): `days` array has 1 entry, so only 1 row renders — this naturally falls back to single-row. No special handling needed.

e. `formatDayLabel(isoDate)`: Use date-fns `format(new Date(isoDate), 'MMM dd')` — e.g., "Nov 14".

f. Build `cellMap` as `new Map<string, HeatmapCell>()` keyed by `"YYYY-MM-DD-H"` for O(1) lookup.
