     - `goodFill`: if `deltaY !== null && Math.abs(deltaY) <= thresholdPx` then `deltaY` else `null`
     - `slouchFill`: if `deltaY !== null && Math.abs(deltaY) > thresholdPx` then `deltaY` else `null`

   - Render with Recharts `ResponsiveContainer` (width="100%" height="100%"):
     ```tsx

       <ComposedChart data={chartData} margin={{ top: 16, right: 16, bottom: 32, left: 48 }}>
         <CartesianGrid strokeDasharray="3 3" stroke={colors.chartGrid} />
         <XAxis
           dataKey="time"
           type="number"
           domain={visibleDomain ?? ['dataMin', 'dataMax']}
           tickFormatter={(t) => { /* format: HH:MM for <24h range, MMM dd HH:MM for multi-day */ }}

             x1={region.startTime}
             x2={region.endTime}

             fillOpacity={0.12}

           />

         <ReferenceLine

           strokeDasharray="6 4"
           strokeWidth={2}
           label={null}

           dataKey="goodFill"

         <Area
           dataKey="slouchFill"
           type="monotone"
           fill={colors.postureSlouch}
           fillOpacity={0.20}
           stroke="none"
           isAnimationActive={!reducedMotion}
           animationDuration={400}
           connectNulls={false}
         />
         {/* Main data line */}

           dataKey="deltaY"
           type="monotone"

           connectNulls={false}
     ```
